# SCT_WD_1

A simple responsive landing page with a sticky navigation header, mobile hamburger menu, smooth scrolling, and accessible focus states.

## Files

- `index.html` — page structure and markup
- `styles.css` — responsive styling, sticky navbar, hover effects, and mobile layout
- `script.js` — mobile nav toggle, sticky header, and active section highlighting

## Run locally

1. Open `index.html` in your browser.
2. Or run a local server from the repo folder, for example:
   - `python -m http.server`
3. Visit `http://localhost:8000`.
